# Install shell completion (MVP)

## Bash
1. Generate:
   `csk completion bash > ~/.csk_completion.bash`
2. Source from `.bashrc`:
   `source ~/.csk_completion.bash`

## Zsh
1. Generate:
   `csk completion zsh > ~/.csk_completion.zsh`
2. Source from `.zshrc`:
   `source ~/.csk_completion.zsh`

## Fish
1. Generate:
   `csk completion fish > ~/.config/fish/completions/csk.fish`
