# 14 — Glossary

- **Engine** — минимальное ядро команд, enforce‑гейтов и генерации.
- **Project State** — долговечные данные проекта (миссии, логи, event log).
- **Local Overlay** — кастомизация (профили, доп. гейты, overrides skills).
- **Module** — единица контекста, имеет свой worktree и задачи.
- **Mission** — большой запрос (multi-module).
- **Milestone** — итерация миссии; в MVP активен только milestone‑1.
- **Task** — работа внутри модуля.
- **Slice** — атомарный шаг (implement + gates).
- **Gate** — проверка (plan/slice/ready).
- **Proof Pack** — набор доказательств прохождения гейтов.
- **Event Log** — append‑only журнал событий (SSOT).
- **PKM** — Project Knowledge Memory (в MVP: runbook facts).
- **Context Bundle** — детерминированный пакет контекста с provenance.
