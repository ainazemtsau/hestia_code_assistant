# Phase plan overview

Фазы идут строго по порядку P00 → P18. Каждая фаза:
- добавляет небольшой инкремент
- заканчивается checklist'ом
- не требует больших рефакторингов предыдущего кода

Рекомендуемый цикл для code assistant:
1) Открой текущую фазу.
2) Реализуй deliverables.
3) Прогони validation checklist.
4) Только после этого переходи к следующей фазе.
