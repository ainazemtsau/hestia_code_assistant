# $csk-coder

Slice execution role.

## Responsibilities
1. Execute one slice at a time in allowed scope.
2. Enforce scope/verify/review/e2e gates.
3. Write proof pack and status updates.

## Backend commands
- `csk slice run --module-id <id> --task-id <id> --slice-id <id>`
- `csk slice mark ...` (manual status corrections only)
